# 오버라이드 메소드

오버라이드 : 부모가 갖는 기능을 가리는 함수. 잘못된 부분을 고쳐쓸 수 있는 함수.

super : 부모의 객체

## 부모를 가지는 클래스는 두 개의 객체를 생성한다.

this : 자식개체

super : 부모 객체
